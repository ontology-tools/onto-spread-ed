# OSE JS App

Vue.js web application components for OntoSpreadEd - the Ontology Spreadsheet Editor.

## Description

This package contains the frontend Vue.js components and application logic for OntoSpreadEd. It includes:

- Spreadsheet editor with Tabulator.js integration
- Ontology visualization using D3.js and Graphviz
- Admin dashboard and settings pages
- Release management interface
- Common UI components

## Installation

This package is private and used as part of the main OntoSpreadEd application.

## Requirements

- Node.js 16+
- Vue 3

## Build

```bash
npm run build
```

For development with watch mode:
```bash
npm run dev
```
